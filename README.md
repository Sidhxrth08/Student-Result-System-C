# Student Result System (C-language)

A simple console-based Student Result System developed in C.

## Features
- Enter student details
- Input marks for 5 subjects
- Calculate total marks
- Calculate percentage
- Display grade
- Display pass/fail status

## Concepts Used
- Functions
- Arrays
- Loops
- Conditional Statements

## How to Run

Compile:
```bash
gcc student_result.c -o result
```

Run:
```bash
./result
```
